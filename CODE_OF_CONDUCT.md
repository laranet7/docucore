# Code of Conduct

DocuCore aims to maintain a respectful, collaborative, and professional environment.

## Expected behavior

- Communicate respectfully and constructively.
- Assume good intent while giving honest technical feedback.
- Focus discussions on the work, not on the person.
- Help keep the project welcoming to contributors with different experience levels.

## Unacceptable behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or exclusionary language
- Repeated disruptive behavior in issues, pull requests, or discussions
- Sharing private or sensitive information without permission

## Reporting concerns

If you experience or witness inappropriate behavior, contact the repository maintainer through GitHub so the issue can be reviewed and handled appropriately.
